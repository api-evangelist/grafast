# Grafast Plans

Grafast is open-source software (MIT licensed) and is free to use. The project is community-funded through sponsorships rather than traditional SaaS subscription plans.

## Open Source — Free

- **License**: MIT
- **Cost**: Free for all uses (personal, commercial, self-hosted)
- **Package**: `npm install grafast`
- **Source**: https://github.com/graphile/crystal

No usage fees, no API keys required, no rate limits imposed by the project itself.

## Sponsorship Tiers (via grafast.org/support/)

The Graphile project sustains Grafast development through voluntary sponsorship. Tiers vary by contribution level and provide perks such as:

### Individual Sponsorship
- Support the maintainer directly via GitHub Sponsors (https://github.com/sponsors/benjie)
- Appropriate for individual developers using Grafast in personal or small projects

### Production Sponsor ($25+/month)
- Suitable for small and medium businesses running Grafast in production
- Recognition as a sponsor on project pages
- Access to priority support channels

### Featured Sponsor (Higher Tiers)
- Recognition as a Featured Sponsor on grafast.org and graphile.org
- Access to discounted consultancy packages
- Formal support contracts available
- One-to-one access to the Graphile core team
- Priority issue resolution

## Commercial Support

Graphile offers optional paid consulting and support arrangements for organizations that need:
- Formal SLAs
- Custom schema design assistance
- Performance tuning for large-scale deployments
- Migration from graphql-js to Grafast

Contact: https://grafast.org/support/
